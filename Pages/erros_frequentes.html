ERROS FREQUENTES

Primeiro possível erro:

1

RECEBA DADOS

O erro "é necessário estar no cliente para iniciar visitas, porém a localização cadastrada no cliente é inexistente ou inválido, ocorre quando o cliente não possui coordenadas cadastradas, para cadastrar as coordenadas clique em "Cancelar" e siga os passos abaixo.

2

RECEBA DADOS

Clique em "Registrar Coordenadas" na tela de cliente, como situado na imagem.

3

RECEBA DADOS

E confirme o registro das coordenadas apertando "Sim".

4

RECEBA DADOS

Se a mensagem mostrada for "Coordenadas registradas com sucesso" a visita pode ser iniciada normalmente.

Segundo possível erro:

5

RECEBA DADOS

O erro "É necessário estar no cliente para iniciar visitas., nesses casos entre em contato com o Suporte Veplex clicando no botão abaixo e encaminhe um print do erro e sua localização atual.
